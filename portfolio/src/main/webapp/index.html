Projects

Save My Wallet

Save My Wallet is an online web application designed to help users manage their subscription payments. The site will calculate both the monthly and annual payments for the user to visually see the true amount that they are paying. The user is able access this information by choosing from a drop down list of over 10+ common subscriptions such as Disney+ and Netflix.

Technology Used: HTML, CSS, Bootstrap, Python, Flask, Javascript, Google Firebase Firestore.
